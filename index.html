CSS Bar

Bienvenue!

CSS Bar est un tutoriel pour apprendre les sélecteurs CSS!

Pour avancer dans le jeu, il suffit d'écrire le bon sélecteur CSS dans la zone "Editeur CSS" ci-dessous

Démo 1 - Règle CSS basique

p {
   color: red;
}

Dans cet exemple, le sélecteur "p" (sélectionne tous les éléments <p>) et leur applique une couleur rouge.

Une aide est présente à droite de l'écran pour chaque sélecteur.

En passant le curseur sur les éléments, tu verras leur structure HTML.

Ok!
Au Secours!
